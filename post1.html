Blog Post 1

Utterances

When you use jemdoc to write a blog post with a valid comment section, we use this extension  —  Utterances.

We do this by just adding this code to the end of your post.

{{
<script src="https://utteranc.es/client.js"
        repo="szl2/szl2.github.io"
        issue-term="pathname"
        theme="github-light"
        crossorigin="anonymous"
        async>
</script>
}}

Although it may show up locally, when your website is host on a server, it will show up on www.

You can view the resulting blog here.
