Version Notation

Every particular note link should have the following fromat:

v#_yyyymmdd

in which # represents number in {0, 1, 2}, which represents "scaned handwritten", "typed pdf", and "web published", respectively. And yyyymmdd represents date this note is updated.

For example, v2_20190101 means this note is in the format of a typed pdf, and last time I edited it was on Jan 1st, 2019.
